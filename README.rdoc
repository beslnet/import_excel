Mini app para carga masiva de datos desde excel a rails.  Acepta CSV (Delimitado por comas), XLS, XLST.

Esto lo hice utilizando la gema Roo-XLS.

Está construido en Rails 4.2.6 y Ruby 2.3.0

Para instalar, clonar y ejecutar las rutinas típicas:

bundle.

rake db:migrate .
